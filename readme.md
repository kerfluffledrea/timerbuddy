

# -[0:0]- Timer Buddy 
A teeny little timer buddy to help you time your time.

<img width="100" height="45" alt="TimerGif" src="https://github.com/user-attachments/assets/2691d393-521d-4880-b34c-e6c8719a6478" />

### Features
- Sick-ass style w/ themes
- Extremely Tactile Operation with Sound Effects (can be disabled)
- Progress Bar
- Smol

## Installation

### Release
1. Download build for your OS from [Releases](https://github.com/Nosler/kanbuddy/releases)
2. Launch it.

### Python 
1. Clone git repository
2. Install Dependencies: `pip install pygame`
3. Run: `python3 timerbuddy.py`

You can just run it with python every time, but if you'd like to build a binary from the script, you can run:

```
pip install pyinstaller
pyinstaller timerbuddy.spec
```
## Operation
- Left-Click & Drag : Edit Time
- Middle-Click : Start / Stop Timer
- Right Click : Context Menu
